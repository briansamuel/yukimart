Manual Test Steps

1Environment Setup

Ensure the server is running and accessible. Click "Load Invoice Page" to begin testing.

  • Server should be accessible at http://yukimart.local
  • Login credentials: yukimart@gmail.com / 123456
  • Invoice data should be available for testing

2Panel Expansion Test

Test the basic functionality of expanding invoice detail panels.

  • Click on any invoice row in the table
  • Verify detail panel appears below the clicked row
  • Check that panel content loads correctly
  • Verify panel has proper styling and layout

3Sticky Position Test

Test the sticky positioning behavior during horizontal scroll.

  • Set viewport to small size (800x600) to force horizontal scrolling
  • Expand a detail panel
  • Scroll the table horizontally using scrollbar or Shift+MouseWheel
  • Verify panel stays fixed at left edge of viewport
  • Confirm table content scrolls behind the panel

4Multiple Panels Test

Test behavior with multiple detail panels open simultaneously.

  • Open first detail panel
  • Open second detail panel
  • Scroll table horizontally
  • Verify both panels maintain sticky positioning
  • Check for any z-index or layout conflicts

5Responsive Design Test

Test sticky positioning at different viewport sizes.

  • Test at desktop size (1200x800)
  • Test at tablet size (1024x768)
  • Test at mobile size (375x667)
  • Verify panel adapts correctly at each size
  • Ensure no horizontal overflow issues

6Tab Interaction Test

Test sticky positioning with tab switching in detail panel.

  • Expand detail panel and scroll table horizontally
  • Switch between tabs in detail panel (if available)
  • Verify panel maintains sticky position during tab switches
  • Check that tab content loads properly
